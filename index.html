Habilidades

  • HTML Basico

    • HTML (HyperText Markup Language) é a base de toda a web. Meu conhecimento abrange os conceitos essenciais para construir a estrutura de uma página, como a organização do conteúdo em seções, a criação de parágrafos e listas, e a correta inserção de links e imagens.

  • CSS Basico

    • CSS (Cascading Style Sheets) é a linguagem que dá vida ao design de um site. Tenho conhecimento básico em estilização para tornar um site visualmente atraente. Sei como aplicar cores, formatar fontes e gerenciar o layout de elementos, usando técnicas como o Flexbox para alinhar e organizar o conteúdo.

  • Python Basico

    • Python é uma linguagem de programação versátil que me permite explorar a lógica por trás dos sistemas. Meus conhecimentos básicos incluem a manipulação de variáveis, o uso de estruturas de controle (como loops e condicionais) e a capacidade de processar diferentes tipos de dados. Estou empolgado para aprofundar esses conhecimentos e aplicá-los em projetos futuros.
